1 Memulai

Login ke Sistem

Sebelum dapat menggunakan fitur apapun, pengguna wajib login terlebih dahulu menggunakan akun yang telah diberikan oleh administrator.

Cara Login

  1. Buka browser (Chrome, Edge, atau Firefox) dan ketik alamat aplikasi di address bar.
  2. Anda akan diarahkan ke halaman Login secara otomatis.
  3. Masukkan Username dan Password yang telah diberikan.
  4. Klik tombol Masuk.
  5. Jika berhasil, Anda akan diarahkan ke halaman Dashboard.
⚠️ Perhatian: Jika login gagal, pastikan username dan password sudah benar (perhatikan huruf kapital). Hubungi administrator jika akun belum aktif.

Logout

Untuk keluar dari sistem, klik nama pengguna di pojok kanan atas, lalu pilih Logout. Selalu logout setelah selesai menggunakan sistem, terutama di perangkat bersama.

2 Navigasi

Dashboard & Navigasi

Dashboard adalah halaman utama yang menampilkan ringkasan data klinik secara cepat.

Informasi di Dashboard

  • Registrasi Hari Ini – jumlah appointment yang terjadwal hari ini.
  • Total Pasien Aktif – jumlah pasien yang terdaftar dan tidak dihapus.
  • Pemasukan Bulan Ini – total pendapatan bulan berjalan.
  • Notifikasi Belum Dibaca – jumlah notifikasi yang masih perlu ditindaklanjuti.
  • Daftar Registrasi Terbaru – tabel beberapa registrasi terakhir yang masuk.
  • Notifikasi Terbaru – ringkasan notifikasi (ulang tahun, pasien tidak aktif, follow-up).

Navigasi Sidebar

Menu utama ada di sidebar kiri. Pada tampilan mobile, klik ikon hamburger (☰) untuk membuka sidebar. Grup menu yang tersedia:

GrupMenu yang Tersedia
UtamaDashboard, Notifikasi, Template Pesan, Registrasi, Kalender
Laporan KeuanganPemasukan, Pengeluaran

ℹ️ Menu yang muncul tergantung hak akses (role) pengguna. Pengguna dengan role terbatas mungkin hanya melihat sebagian menu.

Ikon Notifikasi (Bel)

Di header atas terdapat ikon 🔔 dengan badge merah yang menunjukkan jumlah notifikasi belum dibaca. Klik ikon tersebut untuk melihat daftar notifikasi terbaru.

3 Master Data

Data Pasien

Modul Pasien digunakan untuk mengelola data semua pasien yang terdaftar di klinik. Akses melalui menu Master Data → Pasien.

Melihat Daftar Pasien

Halaman daftar pasien menampilkan tabel berisi semua pasien aktif. Tersedia fitur:

  • Pencarian – ketik nama anak atau nama orang tua di kolom pencarian.
  • Filter Cabang – tampilkan pasien berdasarkan cabang tertentu.
  • Filter Status WhatsApp – tampilkan pasien yang terdaftar/tidak terdaftar di WhatsApp.
  • Pagination – navigasi antar halaman jika data banyak.

Tambah Pasien Baru

  1. Klik tombol + Tambah Pasien.
  2. Isi formulir data pasien (lihat tabel field di bawah).
  3. Klik Simpan.
Nama Anak *Nama lengkap bayi/anak. Wajib diisi.
Tanggal Lahir *Format: YYYY-MM-DD (otomatis dari date picker).
Jenis KelaminPilih: Laki-laki atau Perempuan.
Nama Orang TuaNama ayah/ibu yang mendampingi.
No. WhatsAppNomor HP aktif (format: 08xxxx atau 62xxxx). Sistem akan menggunakannya untuk kirim reminder.
Terdaftar WACentang jika nomor tersebut aktif di WhatsApp.
AlamatAlamat lengkap tempat tinggal pasien.
CabangCabang klinik tempat pasien terdaftar.
Kode PasienKode unik otomatis dibuat sistem (contoh: P-001). Bisa dikosongkan.

Edit Data Pasien

  1. Cari pasien di daftar, lalu klik ikon ✏️ Edit di kolom aksi.
  2. Ubah data yang perlu diperbarui.
  3. Klik Simpan.

Export Data Pasien ke Excel

  1. Di halaman daftar pasien, klik tombol Export Excel.
  2. File Excel akan otomatis terunduh ke komputer Anda.
  3. File berisi semua data pasien yang sedang ditampilkan (sesuai filter aktif).

Import Pasien dari Excel

  1. Klik tombol Import Excel.
  2. Pilih file Excel (.xlsx) yang sudah disiapkan sesuai format template.
  3. Klik Upload & Import.
  4. Sistem akan memvalidasi data dan menampilkan hasil import (berhasil/gagal).
ℹ️ Format Import: File Excel harus memiliki kolom: nama_anak, tanggal_lahir, jenis_kelamin, nama_orang_tua, no_wa, alamat. Header kolom harus tepat.

Tambah Pasien Cepat (dari Form Registrasi)

Saat mengisi formulir Registrasi baru, jika pasien belum terdaftar, Anda dapat membuat data pasien baru langsung dari dropdown melalui tombol "+ Pasien Baru" tanpa harus keluar dari halaman registrasi.

4 Inti Sistem

Registrasi & Appointment

Modul Registrasi adalah inti dari sistem ini. Digunakan untuk mencatat setiap kunjungan/appointment terapi pasien. Akses melalui menu Registrasi.

Status Registrasi

StatusArtiKapan Digunakan
BOOKED Terbooking Appointment baru dibuat, belum konfirmasi
CONFIRMED Dikonfirmasi Pasien sudah konfirmasi akan hadir
COMPLETED Selesai Terapi sudah dilakukan dan selesai
CANCELLED Dibatalkan Appointment dibatalkan

Melihat Daftar Registrasi

Halaman daftar menampilkan semua registrasi. Tersedia filter:

  • Filter Tanggal – filter berdasarkan rentang tanggal kunjungan.
  • Filter Status – tampilkan berdasarkan status (BOOKED, CONFIRMED, dll).
  • Filter Terapis – tampilkan registrasi berdasarkan terapis tertentu.
  • Filter Cabang – untuk klinik multi-cabang.
  • Pencarian – cari berdasarkan nama pasien atau kode registrasi.

Tambah Registrasi Baru

  1. Klik tombol + Registrasi Baru.
  2. Pilih Pasien dari dropdown (ketik untuk mencari). Jika pasien belum ada, klik "+" untuk buat pasien baru.
  3. Pilih Terapis yang akan menangani (opsional – bisa dikosongkan).
  4. Pilih Tanggal Kunjungan menggunakan date picker.
  5. Isi Jam Kunjungan (format HH:MM, contoh: 09:30).
  6. Pilih Jenis Terapi. Harga akan otomatis terisi dari master data.
  7. Jika lebih dari satu jenis terapi (multi-terapi), klik + Tambah Terapi dan ulangi pemilihan.
  8. Atur Biaya Transport terapis (otomatis terisi dari default terapis, bisa diubah).
  9. Centang/hapus centang Ada Transport sesuai kebutuhan.
  10. Pilih Status awal (default: BOOKED).
  11. Isi Catatan jika ada informasi tambahan.
  12. Klik Simpan Registrasi.
✅ Multi-Terapi: Satu registrasi bisa memiliki lebih dari satu jenis terapi. Total harga akan dijumlah otomatis dari semua jenis terapi yang dipilih.

Edit Registrasi

  1. Dari daftar registrasi, klik ikon ✏️ di kolom aksi registrasi yang ingin diubah.
  2. Ubah data yang diperlukan (status, tanggal, terapis, dll).
  3. Klik Simpan Perubahan.
ℹ️ Mengubah Status: Untuk menandai terapi sudah selesai, ubah status dari BOOKED/CONFIRMED menjadi COMPLETED. Saat status COMPLETED, sistem otomatis mencatat data ke laporan keuangan (pemasukan).

Export Registrasi ke Excel

  1. Atur filter terlebih dahulu (tanggal, status, dsb) jika ingin data yang spesifik.
  2. Klik tombol Export Excel.
  3. File Excel berisi semua registrasi sesuai filter yang aktif.
Kolom ExportKode Registrasi, Nama Anak, Tanggal Kunjungan, Jam, Jenis Terapi, Nama Terapis, Harga, Biaya Transport, Total Bayar, Sudah Dibayar, Sisa Tagihan, Status, Cabang.
5 Jadwal Visual

Kalender Appointment

Kalender menampilkan semua appointment dalam tampilan visual kalender bulanan/mingguan. Akses melalui menu Kalender Appointment.

Tampilan Kalender

  • Setiap appointment ditampilkan sebagai blok berwarna di tanggal yang sesuai.
  • Warna blok mewakili status: biru (BOOKED), kuning (CONFIRMED), hijau (COMPLETED), merah (CANCELLED).
  • Klik blok appointment untuk melihat detail (nama pasien, jenis terapi, terapis, jam).

Reschedule via Drag & Drop

  1. Klik dan tahan blok appointment yang ingin dipindahkan.
  2. Seret ke tanggal baru yang diinginkan.
  3. Lepaskan – sistem akan meminta konfirmasi perubahan tanggal.
  4. Klik Ya, Pindahkan untuk mengkonfirmasi.
⚠️ Fitur Reschedule hanya tersedia untuk pengguna dengan permission registrasi_reschedule.

Navigasi Kalender

  • Klik ◀ Prev / Next ▶ untuk berpindah bulan.
  • Klik Today untuk kembali ke hari ini.
  • Pilih tampilan Month / Week / Day di pojok kanan atas kalender.
6 Monitoring

Progress Tracking Terapi

Fitur Progress Tracking memungkinkan terapis mencatat perkembangan kondisi pasien untuk setiap registrasi, lengkap dengan foto sebelum dan sesudah.

Menambah Catatan Progress

  1. Buka halaman daftar Registrasi.
  2. Cari registrasi yang ingin dicatat progressnya.
  3. Klik ikon 📊 Progress atau tombol "Tambah Progress" di kolom aksi.
  4. Isi form berikut:
    Judul *Judul singkat catatan, misal "Sesi 1 - Terapi Sensori".
    CatatanDeskripsi lengkap perkembangan pasien.
    Foto SebelumUpload foto kondisi pasien sebelum terapi.
    Foto SesudahUpload foto kondisi pasien setelah terapi.
  5. Klik Simpan Progress.
💡 Tips: Gunakan fitur ini secara konsisten setiap sesi untuk membantu orang tua pasien melihat perkembangan anak mereka dari waktu ke waktu.
7 Keuangan

Pemasukan (Pendapatan)

Modul Pemasukan digunakan untuk mencatat semua pendapatan klinik. Akses melalui Laporan Keuangan → Pemasukan.

ℹ️ Otomatis dari Registrasi: Ketika status registrasi diubah ke COMPLETED, data pemasukan otomatis tercatat. Namun Anda juga bisa menambahkan pemasukan secara manual untuk pendapatan di luar terapi.

Melihat Daftar Pemasukan

  • Filter Tanggal – tampilkan pemasukan di rentang tanggal tertentu.
  • Filter Cabang – tampilkan pemasukan per cabang.
  • Total pemasukan yang difilter ditampilkan di bagian bawah tabel.

Tambah Pemasukan Manual

  1. Klik + Tambah Pemasukan.
  2. Isi form:
    Tanggal *Tanggal transaksi pemasukan.
    Jumlah *Nominal pemasukan (gunakan titik sebagai pemisah ribuan, misal: 150.000).
    Jumlah BayarJumlah yang sudah dibayarkan (jika berbeda, misal pembayaran cicilan).
    Metode PembayaranTunai, Transfer Bank, QRIS, dsb.
    KeteranganCatatan tambahan tentang transaksi ini.
    RegistrasiOpsional – tautkan ke nomor registrasi jika relevan.
    CabangPilih cabang asal pemasukan.
  3. Klik Simpan.

Edit Pemasukan

Klik ikon ✏️ di baris yang ingin diubah → ubah data → klik Simpan.

8 Keuangan

Pengeluaran

Modul Pengeluaran mencatat semua biaya operasional klinik. Akses melalui Laporan Keuangan → Pengeluaran.

Tambah Pengeluaran Baru

  1. Klik + Tambah Pengeluaran.
  2. Isi form:
    Tanggal *Tanggal pengeluaran terjadi.
    KategoriJenis pengeluaran, misal: Operasional, Gaji, Perlengkapan, Transport Terapis, dll.
    Jumlah *Nominal pengeluaran.
    KeteranganDeskripsi singkat untuk apa pengeluaran ini.
    CabangCabang yang melakukan pengeluaran.
  3. Klik Simpan.
ℹ️ Integrasi Inventory: Pengeluaran dari pemakaian barang inventory akan otomatis tercatat di modul pengeluaran dengan kategori "Pemakaian Barang".

Edit Pengeluaran

Klik ✏️ di baris pengeluaran → ubah data → klik Simpan.

9 Statistik

Rekap Tindakan

Halaman Rekap Tindakan menampilkan statistik jumlah tindakan (registrasi) yang sudah diselesaikan. Akses melalui Rekap & Statistik → Rekap Tindakan.

Cara Menggunakan

  1. Pilih rentang tanggal yang ingin direkap.
  2. Pilih filter Terapis atau Cabang jika diperlukan.
  3. Klik Tampilkan.
  4. Data akan menampilkan jumlah tindakan, jenis terapi yang dilakukan, dan terapis yang menangani.

Data rekap berguna untuk evaluasi kinerja terapis dan mengetahui jenis terapi yang paling sering dilakukan.

10 Akuntansi

Pembukuan & Saldo

Grup menu Pembukuan berisi laporan keuangan komprehensif. Akses dari sidebar bagian Pembukuan.

Total Pendapatan

Menampilkan ringkasan total pendapatan klinik. Dapat difilter per bulan, tahun, atau rentang tanggal. Menampilkan juga rincian per jenis terapi dan per terapis.

Total Pengeluaran

Ringkasan semua pengeluaran klinik dikelompokkan per kategori. Membantu identifikasi pos biaya terbesar.

Rekap Pasien / Terapis

Laporan yang menampilkan detail aktivitas per pasien dan per terapis, termasuk:

  • Jumlah kunjungan pasien dalam periode tertentu.
  • Total pendapatan dari masing-masing pasien.
  • Jumlah sesi yang ditangani per terapis.
  • Total bayar dan status pembayaran (LUNAS / BELUM LUNAS).

Rekap Transport Terapis

Laporan biaya transport yang dibayarkan kepada terapis. Berguna untuk menghitung total biaya transport bulanan dan memverifikasi pembayaran ke masing-masing terapis.

Saldo Akhir

Halaman Saldo Akhir menampilkan ringkasan keuangan keseluruhan:

Total PemasukanJumlah semua pendapatan dalam periode.
Total PengeluaranJumlah semua biaya dalam periode.
Saldo AkhirSelisih Pemasukan – Pengeluaran. Jika positif = Surplus, negatif = Defisit.
💡 Tips: Gunakan halaman Saldo Akhir setiap akhir bulan untuk rekap keuangan bulanan. Sesuaikan filter bulan/tahun sebelum melihat data.
11 Stok Barang

Manajemen Inventory

Modul Inventory digunakan untuk mengelola stok barang klinik (alat terapi, mainan, perlengkapan, dll). Akses dari sidebar bagian Inventory.

Status Stok

StatusKondisi
AMANStok tersedia di atas batas minimum.
RENDAHStok tersedia di bawah atau sama dengan batas minimum. Perlu segera restock.
HABISStok = 0. Tidak ada barang tersedia.

A. Kategori Barang

Sebelum menambah barang, siapkan kategorinya terlebih dahulu. Akses: Inventory → Kategori Barang.

  1. Klik + Tambah Kategori.
  2. Isi Nama Kategori (contoh: Alat Terapi, Mainan Edukatif, Supplies).
  3. Isi Deskripsi (opsional).
  4. Klik Simpan.

B. Daftar Barang

Master data semua barang inventory. Akses: Inventory → Daftar Barang.

Tambah Barang Baru

  1. Klik + Tambah Barang.
  2. Isi form:
    Kode BarangKode unik barang (opsional, sistem bisa auto-generate).
    Nama Barang *Nama lengkap barang.
    Kategori *Pilih dari daftar kategori yang sudah dibuat.
    SatuanUnit satuan barang (pcs, buah, botol, set, dll).
    Stok MinimumBatas stok minimum sebelum alert RENDAH muncul.
    Stok TersediaStok awal saat pertama kali didaftarkan.
    Harga SatuanHarga per unit barang.
    KeteranganCatatan tambahan tentang barang.
  3. Klik Simpan.

C. Stok Masuk

Catat setiap penambahan stok barang (pembelian, penerimaan barang baru). Akses: Inventory → Stok Masuk.

  1. Klik + Tambah Stok Masuk.
  2. Pilih Barang dari dropdown.
  3. Isi Jumlah barang yang masuk.
  4. Isi Tanggal penerimaan barang.
  5. Isi Harga Satuan pembelian.
  6. Isi Keterangan (supplier, nomor invoice, dll).
  7. Klik Simpan. Stok barang otomatis bertambah.

D. Pemakaian Barang

Catat setiap pemakaian barang untuk sesi terapi. Akses: Inventory → Pemakaian Barang.

  1. Klik + Tambah Pemakaian.
  2. Pilih Barang yang dipakai.
  3. Isi Jumlah yang digunakan.
  4. Pilih Tanggal pemakaian.
  5. Tautkan ke Registrasi jika pemakaian terkait sesi terapi tertentu (opsional).
  6. Isi Keterangan.
  7. Klik Simpan. Stok barang otomatis berkurang. Biaya pemakaian otomatis tercatat sebagai pengeluaran.
⚠️ Perhatian: Pastikan jumlah pemakaian tidak melebihi stok tersedia. Sistem akan memperingatkan jika stok tidak mencukupi.

E. Laporan Inventory

Halaman laporan menampilkan ringkasan kondisi inventory saat ini, termasuk:

  • Daftar barang dengan status stok (AMAN/RENDAH/HABIS).
  • Riwayat keluar masuk barang dalam periode tertentu.
  • Total nilai inventory.
  • Barang yang perlu segera di-restock (stok rendah/habis).
12 Reminder

Notifikasi Otomatis

Sistem secara otomatis membuat notifikasi untuk membantu staff memantau kondisi pasien. Akses dari menu Notifikasi atau klik ikon 🔔 di header.

Jenis Notifikasi

JenisKapan MunculTindakan

Mengelola Notifikasi

Tandai Semua Sudah Dibaca

  2. Semua notifikasi yang belum dibaca akan ditandai sekaligus.

Generate Notifikasi Manual

Notifikasi biasanya di-generate otomatis setiap hari. Jika ingin memperbarui sekarang:

  1. Klik tombol 🔄 Generate Notifikasi di halaman Notifikasi.
  2. Sistem akan memeriksa kondisi semua pasien dan membuat notifikasi baru.
13 Komunikasi

Template Pesan WhatsApp

Fitur Template Pesan memungkinkan pembuatan format pesan standar yang dapat digunakan berulang kali untuk komunikasi dengan orang tua pasien. Akses dari menu Template Pesan.

Placeholder yang Tersedia

Gunakan placeholder berikut dalam teks template. Saat pesan dikirim, placeholder akan otomatis diganti dengan data yang sebenarnya.

PlaceholderDiganti dengan
{nama_anak}Nama anak/pasien
{nama_orang_tua}Nama orang tua
{tanggal}Tanggal kunjungan
{jam}Jam kunjungan
{jenis_terapi}Jenis terapi yang dijadwalkan
{nama_terapis}Nama terapis yang menangani
{nama_klinik}Nama klinik/cabang

Membuat Template Baru

  1. Klik + Tambah Template.
  2. Isi Nama Template (contoh: "Reminder Kunjungan H-1").
  3. Isi Isi Pesan menggunakan placeholder di atas.
    Contoh: "Halo Bapak/Ibu {nama_orang_tua}, kami mengingatkan bahwa {nama_anak} memiliki jadwal terapi besok, {tanggal} pukul {jam}. Terapis: {nama_terapis}. Mohon hadir tepat waktu."
  4. Klik Simpan.

Edit / Hapus Template

  • Klik ✏️ untuk mengedit template yang sudah ada.
  • Klik 🗑️ untuk menghapus template (muncul konfirmasi sebelum dihapus).
14 Komunikasi

Kirim Reminder WhatsApp

Untuk mengirim pesan pengingat ke orang tua pasien melalui WhatsApp langsung dari sistem.

Cara Mengirim Reminder

  1. Buka halaman daftar Registrasi.
  2. Cari registrasi yang ingin dikirimkan reminder.
  3. Klik ikon 💬 WhatsApp di kolom aksi.
  4. Pilih Template Pesan yang ingin digunakan (pesan sudah terisi otomatis dengan data pasien).
  5. Periksa dan edit pesan jika perlu.
  6. Klik Kirim via WhatsApp.
  7. Browser akan membuka WhatsApp Web/App dengan pesan sudah terisi. Klik tombol kirim di WhatsApp.
ℹ️ Syarat: Pasien harus memiliki nomor WhatsApp yang terisi dan opsi "Terdaftar WA" harus dicentang pada data pasien.

16 Master Data

Data Terapis

Modul Terapis menyimpan data semua terapis yang bekerja di klinik. Akses melalui Master Data → Terapis.

Tambah Terapis Baru

  1. Klik + Tambah Terapis.
  2. Isi form:
    Nama Terapis *Nama lengkap terapis.
    No. HPNomor handphone terapis.

17 Master Data

Jenis Terapi

Modul ini menyimpan daftar semua jenis terapi yang tersedia beserta harganya. Akses melalui Master Data → Jenis Terapi.

Tambah Jenis Terapi Baru

  1. Klik + Tambah Jenis Terapi.
  2. Isi form:
    Nama Terapi *Nama jenis terapi (contoh: Terapi Wicara, Terapi Sensori Integrasi, Terapi Okupasi).
    Harga *Biaya untuk satu sesi terapi ini.
    Usia MinimumBatas minimum usia pasien yang disarankan (dalam bulan).
    Usia MaksimumBatas maksimum usia pasien yang disarankan (dalam bulan).
  3. Klik Simpan.
ℹ️ Harga Otomatis: Saat memilih jenis terapi di form Registrasi, harga akan otomatis terisi sesuai data master. Anda tetap bisa mengubah harga di form registrasi jika ada penyesuaian.

Edit / Nonaktifkan Jenis Terapi

  • Klik ✏️ untuk mengubah nama atau harga terapi.
  • Klik 🗑️ untuk menonaktifkan jenis terapi yang sudah tidak digunakan.

Manajemen User

Hanya Superadmin / Owner yang dapat mengelola akun pengguna. Akses melalui menu Pengaturan → User.

Melihat Daftar User

Tabel menampilkan semua akun pengguna beserta informasi: username, nama lengkap, cabang, role, dan status aktif.

Tambah User Baru

  1. Klik + Tambah User.
  2. Isi form:
    Username *Nama login unik. Gunakan huruf kecil tanpa spasi (contoh: budi.staff).
    Nama LengkapNama asli pengguna.
    EmailAlamat email pengguna (opsional).
    Password *Password awal. Minta pengguna menggantinya setelah login pertama.

19 Administrasi

Role & Hak Akses

Sistem menggunakan Role-Based Access Control (RBAC) – setiap pengguna memiliki role yang menentukan menu dan aksi apa saja yang bisa mereka lakukan. Akses melalui Pengaturan → Roles.

Role Bawaan Sistem

RoleDeskripsi
Superadmin / OwnerAkses penuh ke semua fitur tanpa batasan. Tidak dapat diedit permissionnya.
AdminAkses ke sebagian besar fitur manajemen, termasuk keuangan dan laporan.
TerapisAkses terbatas – hanya bisa melihat jadwal sendiri dan menambah progress tracking.
ResepsionisAkses ke registrasi dan data pasien, tidak ke laporan keuangan.

Membuat Role Kustom

  1. Klik + Tambah Role.
  2. Isi Nama Role.
  3. Centang permission yang ingin diberikan kepada role ini dari daftar yang tersedia.
  4. Klik Simpan.

Generate Role Default

Jika ingin mereset role ke default bawaan sistem, klik tombol "Seed Default Roles". Ini akan membuat ulang role Admin, Terapis, dan Resepsionis dengan permission standar.

21 Keamanan Data

Backup Database

Fitur Backup Database memungkinkan administrator membuat salinan data untuk mencegah kehilangan data. Akses melalui Pengaturan → Backup.

Membuat Backup

  1. Klik tombol 🗄️ Buat Backup Sekarang.
  2. Sistem akan memproses backup (progres ditampilkan secara real-time).
  3. Setelah selesai, file backup tersedia untuk diunduh.
  4. Klik Unduh untuk menyimpan file backup ke komputer Anda.

Riwayat Backup

Halaman Backup menampilkan daftar backup yang pernah dibuat, termasuk:

  • Nama file backup.
  • Status: PENDING, IN PROGRESS, COMPLETED.
  • Ukuran file.
  • Tanggal dan waktu backup dibuat.
  • Siapa yang membuat backup.
⚠️ Rekomendasi: Lakukan backup secara rutin minimal seminggu sekali, dan simpan file backup di lokasi yang aman (cloud storage, hard disk eksternal). Jangan simpan backup hanya di komputer yang sama dengan server.
22 Keamanan

Ganti Password

Setiap pengguna dapat mengganti password-nya sendiri. Akses melalui menu di pojok kanan atas → Ganti Password, atau langsung ke Pengaturan Akun → Ganti Password.

Cara Ganti Password

  1. Isi Password Lama (password yang sedang digunakan saat ini).
  2. Isi Password Baru yang diinginkan.
  3. Isi ulang Password Baru di kolom Konfirmasi Password Baru.
  4. Klik Ganti Password.
  5. Anda akan diminta login ulang dengan password baru.
💡 Tips Keamanan Password:
  • Gunakan minimal 8 karakter.
  • Kombinasikan huruf besar, huruf kecil, angka, dan simbol.
  • Jangan gunakan tanggal lahir atau nama sebagai password.
  • Ganti password secara berkala (minimal 3 bulan sekali).
